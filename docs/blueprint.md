# **App Name**: TeamSocial

## Core Features:

- Post Scheduler: Cross-platform post scheduling with a drag-and-drop interface.
- Analytics Dashboard: A unified analytics dashboard to visualize and report social media performance across all platforms.
- Unified Inbox: Combined messaging inbox for managing communications across all social media channels.
- User Management: User and team management features for assigning roles and permissions.
- AI Content Assistant: AI-powered content suggestion tool that analyzes trends and generates engaging copy variations.
- Content Calendar: A calendar view to see the scheduled posts.

## Style Guidelines:

- Primary color: Modern blue (#4285F4), representing trust and professionalism. This hue creates a solid, dependable feeling appropriate for business-oriented software. 
- Background color: Very light blue (#E3F2FD), providing a clean and airy feel.
- Accent color: A slightly purple blue (#5C6BC0) to complement the primary blue and add depth.
- Body and headline font: 'Inter', a sans-serif font known for its clean, modern appearance and readability.
- Simple, line-based icons for a clean and modern interface.
- Clean, well-organized layout with clear information hierarchy for easy navigation.
- Subtle transitions and animations to enhance user experience without being distracting.
